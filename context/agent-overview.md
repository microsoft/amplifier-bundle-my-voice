# Voice System Agents

Two specialized agents for voice preservation work:

| Agent | Use When | Has |
|-------|----------|-----|
| `my-voice:voice-analyst` | Building/updating voice profiles, analyzing sessions | Session parsing, pattern extraction algorithms |
| `my-voice:message-tuner` | Cleaning up messages, adapting for audiences | Voice profile, style matching algorithms |

## Quick Reference

**"Build a voice profile"** → `my-voice:voice-analyst`
**"Clean up this message"** → `my-voice:message-tuner`

See `my-voice:context/delegation-policy.md` for full delegation rules.
